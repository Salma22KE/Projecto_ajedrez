# Projecto_ajedrez
Projecto_ajedrez
Este proyecto es un **juego de ajedrez online** con interfaz web y soporte para partidas en tiempo real entre dos jugadores, utilizando **Node.js**, **Express** y **Socket.io**.

## Estructura del proyecto
ajedrez-proyecto/
├── public/
│ ├── index.html # Archivo HTML principal
│ ├── main.js # Lógica del frontend
│ └── style.css # Estilos del proyecto
├── server/
│ └── index.js # Servidor Node.js
├── package.json # Dependencias y scripts del proyecto
└── package-lock.json # Bloqueo de versiones de dependencias

## Requisitos

- Node.js v18+  
- npm (Node Package Manager)  

## Instalación

1. Clona el repositorio:

```bash
git clone <https://github.com/Salma22KE/Projecto_ajedrez.git>
cd ajedrez-proyecto
```
#Instalar las dependencias:
npm install

Ejecuta el servidor:
cd server
node index.js
